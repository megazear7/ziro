# Ziro

A library of web components

## Using

```bash
npm install ziro
```

```js
import 'ziro';
```

```html
<ziro-panel></ziro-panel>
```

## Contributing

### Install

```
npm install
```

### Serve

```
npm start
```

Open [localhost:8000/dev](http://localhost:8000/dev)

### Build

```
npm run build
```

Then push to the master branch.

### Documentation

#### Serve

```
npm run docs:serve
```

Open [localhost:3000](http://localhost:3000)

#### Build

```
npm run docs:build
```
